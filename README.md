# 💼 Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML5, CSS3, and JavaScript** to showcase my professional profile, technical skills, education, projects, and contact information.

## 📌 About the Project

This portfolio was developed as part of a university CV Portfolio project. It serves as an online representation of my resume and demonstrates my front-end web development skills through a clean, responsive, and user-friendly design.

The website highlights my background in Computer Science and Informatics while showcasing projects, technical skills, and career aspirations in software development.

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Professional landing page with hero section
- About Me section
- Technical Skills section
- Education timeline
- Projects showcase
- Experience section
- Contact section
- Smooth scrolling navigation
- Modern animations and hover effects
- Social media links

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Primrose01/portfolio.git
```

2. Navigate to the project folder

```bash
cd portfolio
```

3. Open `index.html` in your browser.

No additional installations or dependencies are required.
